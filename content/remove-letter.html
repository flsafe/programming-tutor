Write a function that takes two arguments. The first argument is a char 'c'. The second is a char array 'str'. The function deletes all instances of 'c' in 'str'.

For example: 


remove_char('c', 'acacac') => 'aaa'

remove_char('c', 'aaaac')  => 'aaaa'
	
remove_char('c', 'abab')   => 'abab'

Notice that the function 'remove_char' returns 'void' and modifies the 'str' input array in place.

Hint 1:

Most people on their first try come up with an algorithm like this:


for each letter L in str
 if L equals C
  shift everything to the right of L to the left by one

This is very inefficient and is O(n^2) in the worst case. You can do better!

In fact how could you do this in linear time?
